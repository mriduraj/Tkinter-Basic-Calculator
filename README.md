# Tkinter-Basic-Calculator
My First Tkinter Calculator

Hey everyone, this is my very first Python GUI project. I built a simple, working calculator using the Tkinter library to practice what I've learned so far.

I started learning Python just two months ago, so I'm still pretty new to programming. This project is just the beginning for me. As I keep learning and getting better at Python, I plan to come back to this repository and update the calculator to make it look way more modern and add a lot more useful features. 

Right now, it handles all the basic stuff like addition, subtraction, multiplication, and division. It also has a fully working clear function to wipe the screen and basic error handling so the app doesn't crash if you type something wrong. 

How to run it
Make sure you have Python installed, then just open your terminal in this folder and run:
python main.py

Things I want to add in the future
Next up, I want to overhaul the UI because the default Tkinter look is kind of old-school. I also want to add a proper dark mode toggle, a backspace button to delete single numbers, and eventually some scientific calculator functions like square roots and powers once I figure out the logic for them.
